# n8n AI Email Triage — Gmail → Claude AI → Sheets → Telegram

Automatically classify, prioritize, and summarize every incoming email using AI — then log results to Google Sheets and fire instant Telegram alerts for hot leads.

**Live demo:** [Watch on YouTube](https://youtu.be/u2z3gkxVXIM)
**Portfolio:** [Upwork case study](https://www.upwork.com/freelancers/etuannv?p=2069729811119497216)

---

## The Problem

A busy founder's inbox mixes hot leads, recruiters, and spam. Important opportunities get buried for hours — or missed entirely.

## The Solution

A 6-node n8n workflow that runs every minute:

```
Gmail Trigger
     ↓
Normalize Email
     ↓
Claude AI — classify + prioritize + summarize + draft reply
     ↓
Parse JSON
     ↓
Google Sheets — log everything
     ↓
Telegram — instant alert for high-priority emails
```

Every email is classified into one of five categories:
`hot_lead` · `question` · `recruiter` · `spam` · `other`

Each with a priority score (`high` / `medium` / `low`), a 1-sentence summary, and a suggested reply draft — all generated by Claude AI in under 5 seconds.

## Results

- Every inquiry triaged in under 60 seconds
- Zero manual sorting
- Hot leads surface the moment they arrive — with a reply draft ready to send
- Cost: under $0.001 per email (Claude Haiku)

## Setup

### Prerequisites
- n8n instance (self-hosted or cloud)
- Anthropic API key
- Google Cloud project with Gmail API + Sheets API enabled
- Telegram bot token (via @BotFather)

### Install

1. Clone this repo:
```bash
git clone https://github.com/etuannv/n8n-ai-email-triage.git
```

2. Copy the environment template:
```bash
cp .env.example .env
# Fill in your API keys
```

3. Import the workflow into n8n:
- Go to **Workflows → Import from File**
- Select `workflow/ai-email-triage.json`

4. Set up credentials in n8n:
- **Gmail OAuth2** — connect your Gmail account
- **HTTP Header Auth** — name: `x-api-key`, value: your Anthropic API key
- **Google Sheets OAuth2** — connect your Google account
- **Telegram API** — paste your bot token

5. Create a Google Sheet with these exact column headers:
```
receivedAt | senderEmail | subject | category | priority | language | summary | suggested_reply
```

6. In the **Telegram Alert** node, replace `REPLACE_TELEGRAM_CHAT_ID` with your chat ID (get it from @userinfobot).

7. Activate the workflow — it will poll Gmail every minute automatically.

### Environment Variables

```bash
# .env.example
ANTHROPIC_API_KEY=your_key_here
# Other secrets are stored in n8n credentials (encrypted)
```

## How It Works

### AI Prompt Design

The key is forcing Claude to return **strict JSON** with no preamble or markdown fences:

```
Classify this email. Respond with ONLY a valid JSON object with exactly
these keys: category, priority, language, summary, suggested_reply.
Output nothing except the JSON object.
```

This makes the output reliably parseable by the next node without brittle regex.

### Error Handling

- The Parse JSON node strips accidental markdown fences before parsing
- A separate Error Handler workflow sends a Telegram alert if anything fails
- Gmail polling handles API rate limits automatically via n8n's built-in retry
